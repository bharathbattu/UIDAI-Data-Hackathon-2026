# Data Source

## Datasets Used

1. **Aadhaar Enrolment Dataset** (500K+ records)
2. **Aadhaar Demographic Update Dataset** (500K+ records)
3. **Aadhaar Biometric Update Dataset** (500K+ records)

**Source:** UIDAI via event.data.gov.in
**Period:** March 2025 - October 2025 (7 months)
**License:** OGD Platform Terms of Use (Creative Commons By 4.0)

## Download Instructions

1. Visit: https://event.data.gov.in/challenge/uidai-data-hackathon-2026/
2. Download all 3 ZIP files:
   - api_data_aadhar_enrolment.zip
   - api_data_aadhar_demographic.zip
   - api_data_aadhar_biometric.zip
3. Extract to data/raw/ folder (if running locally)

**Note:** Raw data files (.csv, .zip) are NOT committed to Git due to file size.
For analysis, datasets are loaded directly from ZIP files in Google Colab.
